# Platform Badge Images

Please place your platform logo images in this folder with the following names:

- `netflix.png` - Netflix logo
- `disneyplus.png` - Disney+ logo  
- `skyshowtime.png` - SkyShowtime logo
- `hbomax.png` - HBO Max logo
- `voyo.png` - Voyo logo

The application will automatically use these local images if they exist, otherwise it will fall back to external URLs.
