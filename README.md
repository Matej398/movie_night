# Movie Night

A personal movie library dashboard to track movies you want to watch and have watched.

## Features
- **Local Database**: Stores your movies in a persistent MySQL database.
- **User Auth**: Secure login and registration.
- **Smart Search**: Auto-fills movie details from IMDb.
- **Streaming Links**: Direct links to Netflix, Disney+, etc.
- **Optimized Performance**: Lazy loading, image optimization, and smooth transitions.

## Setup
1. Import `database.sql` into your MySQL database.
2. Configure `api/db.php` with your credentials.
3. Host on a PHP-enabled web server.
